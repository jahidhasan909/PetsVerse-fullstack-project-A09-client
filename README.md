
* * PETSVERSE - Pet Adoption Platform :
PETSVERSE হলো একটি Full Stack Pet Adoption Platform যেখানে User বিভিন্ন Pets দেখতে, Adoption Request পাঠাতে এবং নিজের Request Manage করতে পারে।
এই Project টি MERN Stack ব্যবহার করে তৈরি করা হয়েছে।

🔗 Live Link:    


* Key Features
1. বিভিন্ন Pets Browse করা যায়
2. Pet Search, Filter & Sorting করা যায়
3. User Adoption Request পাঠাতে পারে
4. proxy js use private route
5. Pet Add, Update এবং Delete করা যায়
6. User নিজের Adoption Request দেখতে পারে
7. Pet Owner Adoption Approve / Reject করতে পারে
8. Fully Responsive Design (Mobile, Tablet, Desktop)
9. Unauthorized User Private Route Access করতে পারে না
10. HTTPOnly Cookie Based Authentication ব্যবহার করা হয়েছে (jwt)
11.  Search & Filter Features MongoDB Operator ব্যবহার করা হয়েছে:$regex,$in
